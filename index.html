💡 Conseils & Astuces
🖼️
Photos de joueurs : PNG avec fond transparent pour un rendu parfait dans les cercles. JPG fonctionne aussi.
📸
Téléchargement : Le bouton Télécharger génère un PNG 3× (haute résolution) — bien mieux qu'une capture d'écran manuelle pour WhatsApp.
💾
Données persistantes : Tout est sauvegardé automatiquement dans le navigateur (localStorage). Ferme et réouvre le fichier → tout est restauré.
⌨️
Point médian · sur mobile : maintiens appuyé la touche . pour faire apparaître · , ou utilise | ou : comme alternatives.
🎨
Classement : Pour surligner ton équipe, tape exactement le même nom que dans la liste (ou une partie du nom suffit).
🔁
Réutilisation : Garde le fichier HTML sur ton téléphone ou PC. Ouvre-le dans Chrome, Safari ou Firefox — fonctionne hors connexion après le 1er chargement.
📐
Formats optimaux : But (portrait) • Classement/Convoqués (auto-hauteur) • Stats/MOTM (carré) • Tous optimisés pour WhatsApp 1080px.
